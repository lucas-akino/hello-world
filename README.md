# This is test of github

> Start a repository
> ```
>   git init
> ```

> Add file 
> ```
>   git add <file> 
>   git add . // All file
> ```

> make a commit
> ```
>   git commit -m "messaje"
> ```

> make a branch
> ```
>   git branch <newBranch>
> ```

> Show all branch
> ```
>   git branch
> ```

> Change branch
> ```
>   git checkout <nameBranch>
> ```

> Add url remote repository 
> ```
>   git remote add origin https://github.com/user/github-repository.git
> ```

> Push repository
> ```
>   git push -u origin master
> ```
